# Cronómetro en C# - Proyecto Taller 1

Este proyecto es una aplicación simple de cronómetro desarrollada en C# utilizando Windows Forms. El cronómetro tiene la funcionalidad de contar horas, minutos y segundos, con opciones para iniciar, pausar y reiniciar. Además, incluye un modo claro y un modo oscuro que permite al usuario cambiar la apariencia de la interfaz.

## Características
- Iniciar, pausar y reiniciar el cronómetro.
- Cambio entre modo claro y oscuro.
- Interfaz gráfica sencilla con botones y pantalla de tiempo.

## Requisitos
- .NET Framework 4.5 o superior.
- Visual Studio o cualquier otro editor compatible con C#.

## Instalación

1. Clona este repositorio o descarga el código fuente.
2. Abre el proyecto en Visual Studio.
3. Compila y ejecuta el proyecto.
4. ¡Disfruta usando el cronómetro!

## Uso

- **Iniciar:** Inicia el cronómetro.
- **Pausar:** Detiene el cronómetro en su estado actual.
- **Reiniciar:** Reinicia el cronómetro a cero.

Además, puedes alternar entre **modo claro** y **modo oscuro** haciendo clic en el botón de "Dark Mode" en la interfaz.
